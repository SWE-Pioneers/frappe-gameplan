# Gameplan

Gameplan is an async-first collaboration product organized around communities and spaces.

## Language

**Community**:
The top-level collaboration scope that groups people and spaces.
_Avoid_: Team, Category

Use community language in app-layer code for current product concepts. Persisted schema may still use existing technical names until a schema migration is explicitly planned.

**Space**:
A focused area inside a community where discussions, tasks, and pages live.
_Avoid_: Project
